### 4. **Add Explicit E-E-A-T Signals**

**Why:** E-E-A-T (Experience, Expertise, Authoritativeness, Trustworthiness) is the **most critical factor** for AI citations, especially for Gemini.

**Problem:** JetWash is an **anonymous lead-gen site** with no real business identity.

**Creative Solutions for Anonymous Sites:**

**Experience Signals:**
- Add "industry insights" based on aggregated data ("In our experience working with Surrey homeowners...")
- Include before/after scenario descriptions
- Use first-person language sparingly but strategically

**Expertise Signals:**
- Cite industry standards (British Cleaning Council, UKCPI guidelines)
- Reference equipment specifications ("Professional-grade Honda GX engines...")
- Link to UK building/property maintenance standards

**Authority Signals:**
- Link to authoritative sources (.gov.uk, .ac.uk, industry bodies)
- Create comprehensive guides that demonstrate depth
- Build internal topical authority through pillar pages (ALREADY DONE ✅)

**Trust Signals:**
- Show last updated dates prominently
- Add privacy policy (ALREADY DONE ✅)
- Include service area transparency
- Display callback guarantee prominently (ALREADY DONE ✅)

**Implementation Priority:**
1. Add "last updated" timestamps to all pages
2. Include 2-3 authoritative external links per guide
3. Reference UK-specific standards/regulations
4. Add equipment/method descriptions for expertise
