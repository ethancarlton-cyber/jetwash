# JetWash Link Building Strategy & Outreach Guide

## Overview

This document provides templates, strategies, and a target list for building quality backlinks to the JetWash website. Local links are crucial for establishing authority and improving local search rankings.

### Listing Submission Tips

1. **Use consistent NAP** (Name, Address, Phone) on ALL listings
2. Upload the same high-quality photos to each directory
3. Use the same business description (adapt length as needed)
4. Verify listings via phone/email when possible
5. Check back monthly to respond to reviews

## Phase 3: Local Link Opportunities

### Charity & Community Sponsorship

**Strategy:** Offer free or discounted services to local charities in exchange for a mention/link on their website.

**Targets:**
- Local churches (offer free path/car park cleaning)
- Community centres
- Scout/Guide groups
- Local sports clubs

**Outreach Template:**

```
Subject: Community Partnership Offer - Free Pressure Washing

Dear [Organization Name],

I run a local pressure washing company based in Redhill, and I'd like to offer our services to support your organization.

We would be happy to provide [free/heavily discounted] cleaning of your [car park/paths/outdoor areas] as part of our community support programme.

In return, we would appreciate a mention on your website or newsletter with a link to our site: https://jetwash-psi.vercel.app

This is a genuine offer to give back to the local community - no strings attached!

Would you be interested in discussing this further?

Best regards,
[Name]
Same Day JetWash Reigate
```

### Supplier/Partner Links

Contact pressure washing equipment suppliers about:
- Case study features
- Customer testimonials on their site
- "Professionals who use our equipment" pages

## Tracking & Management

### Link Building Spreadsheet Columns

Track your outreach in a spreadsheet with these columns:

| Column | Description |
|--------|-------------|
| Target Site | Website name |
| URL | Website URL |
| Contact | Name/email of contact |
| Type | Directory / Guest Post / Sponsorship / etc. |
| Date Contacted | When you reached out |
| Follow-up Date | When to follow up |
| Status | Pending / Accepted / Rejected |
| Link Live | Yes/No with date |
| Link URL | URL of the page with your link |
| DA | Domain Authority (if known) |
| Notes | Any relevant notes |

### Monthly Tasks

1. **Week 1:** Submit to 5 new directories
2. **Week 2:** Send 3 outreach emails (guest posts, sponsorships)
3. **Week 3:** Follow up on previous outreach
4. **Week 4:** Check existing links still work, monitor new opportunities

## Important Notes

### Do's
- Always provide genuine value in outreach
- Personalize every email
- Follow up once after 1-2 weeks
- Build relationships, not just links
- Prioritize relevant, local links over volume

### Don'ts
- Never buy links
- Don't use automated outreach tools
- Don't spam the same site multiple times
- Don't exchange links (link swaps)
- Don't use exact-match anchor text excessively
