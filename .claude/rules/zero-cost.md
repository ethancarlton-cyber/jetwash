# Zero-Cost Mandate

This project has a strict zero-cost requirement until ROI is proven.

## Rules

1. **NEVER suggest paid services** - Always use free tiers
2. **NEVER suggest custom domains** - Use free subdomains until £200+ revenue
3. **NEVER suggest paid ads** - Organic traffic only until proven

## Approved Free Services

- Hosting: Vercel (FREE unlimited)
- Forms: Web3Forms (FREE 250/month)
- Analytics: PostHog EU Cloud (FREE 1M events/mo + 5K replays/mo)
- SEO: Google Search Console (FREE)
- Repo: GitHub (FREE)

## When Asked About Alternatives

Always check:
1. Does it have a free tier?
2. Is the free tier sufficient for our needs?
3. No credit card required?

If any answer is "no", do not suggest it.
